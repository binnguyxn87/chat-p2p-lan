# Chat P2P LAN

Đồ án Chat P2P mã hóa RSA + AES trong mạng LAN.

## Cấu trúc thư mục
- `network/` - Module Networking (phụ trách: Nguyễn Trường Nhân)
- `crypto/` - Module Bảo mật RSA+AES (phụ trách: Tống Quang Hưng)
- `ui/` - Module Giao diện (phụ trách: Lương Viết Hoàn Thiện)
- `docs/` - Tài liệu, báo cáo
